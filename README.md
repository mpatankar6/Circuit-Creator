# Circuit Creator
Web app in which you can create a complex circuit of resistors and recieve a rendered schematic and a detailed number sheet. 

# Running
```
py app.py
```
Navigate to `127.0.0.1:5000` or `localhost:5000` in your browser.

# Tools Used
- Flask
- Schemdraw 
- Jinja2/JS/CSS
